# Changelog

All notable changes to GridGuard-Pi5 will be documented here.

## [1.0.0] - 2024-02-10

### Added
- Initial release of GridGuard-Pi5
- Real-time electrical monitoring (voltage, current, power)
- Multi-circuit support (up to 12 channels)
- Fault detection (overload, voltage faults)
- Energy consumption tracking
- Cost calculation
- Alert system (email, SMS, local)
- Web dashboard
- REST API
- SQLite database
- Machine learning ready
- Comprehensive documentation

### Features
- ACS712/SCT-013 current sensor support
- ZMPT101B voltage sensor support
- ADS1115 16-bit ADC support
- 100-1000 Hz sampling rate
- Power factor calculation
- THD analysis ready
- MQTT integration ready
- Home Assistant integration

### Safety
- Extensive safety warnings
- Professional installation required
- Simulation mode for testing
- Comprehensive error handling
