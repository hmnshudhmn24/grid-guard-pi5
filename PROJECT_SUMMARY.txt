## ğŸš€ Quick Start

1. **Extract**: `tar -xzf gridguard-pi5.tar.gz && cd gridguard-pi5`
2. **Install**: `sudo ./install.sh`
3. **Configure**: Edit `config/config.yaml`
4. **Calibrate**: `python3 calibrate.py` (with electrician)
5. **Test**: `python3 main.py --test`
6. **Start**: `sudo systemctl start gridguard`

Dashboard: **http://raspberrypi.local:8080**

## ğŸ”§ Hardware Requirements

**Required:**
- Raspberry Pi 5 (4GB or 8GB)
- ADS1115 16-bit ADC (I2C)
- Current sensors (ACS712 or SCT-013)
